## 오늘 배운 것
    1. print() 관련 심화내용 
    1-1. 문자열과 숫자의 구분, +와 ,의 차이, str()함수. 
    1-2. ,로 이어주면 띄어쓰기 +로 이어주면 띄어쓰기 없이 이어준다.
    1-3. str() = 숫자를 문자열로 바꿔주는 함수. 문자열+문자열 구조일떄 숫자를 반드시 바꿔야 오류가 발생하지 않음.
    1-4. print() 괄호 내에선 줄을바꿔 입력문을 작성해도 한 줄로 나옴.

    2. 바이브 코딩 : 암호생성기 및 브리프 생성기를 하나의 CLI 프로그램(.py)안에 구현하기.
    2-1. argparse 라이브러리 활용: 터미널에서 명령행 인자(pw, brief)를 받아 서로 다른 기능을 수행하는 구조를 이해함.
    2-2. 사용자 편의성 개선 (nargs='+'):
        기존에는 긴 문장을 입력할 때 반드시 큰따옴표("")로 감싸야 했으나, nargs='+' 옵션을 통해 띄어쓰기가 포함된 여러 단어를 하나의 리스트로 받아 합치는 방법을 배움.
    2-3. 조건부 비밀번호 생성 로직 (Logic Design):
        강제성 부여: random.choice()를 사용해 대문자 1개와 특수문자 1개를 먼저 확정적으로 뽑는 법.
        중복 제거: random.sample()을 사용하여 한 번 뽑힌 문자가 다시 나오지 않게 하는 법.
        순서 섞기: random.shuffle()을 활용해 특정 문자가 항상 앞에 오지 않도록 무작위성을 부여하는 법.
    2-4. 리스트 조작 (.append() vs +=):
        단일 아이템을 추가할 때는 .append()가 메모리 효율과 가독성 면에서 유리하다는 점을 배움.
    2-5. 요약 방식의 이해:
        현재 구현한 방식은 문맥을 파악하는 '생성적 요약(AI)'이 아니라, 정해진 규칙에 따라 문장을 골라내는 '추출적 요약'임을 이해함.
